# Cardio Good Fitness Customer Segmentation and Product Analysis
End-to-end customer segmentation and product analysis using Python (Pandas) and Power BI to support data-driven marketing decisions.

## Project Overview
This project analyses customer purchasing behaviour across different treadmill models to support data-driven marketing and product positioning decisions.

## Business Problem
A fitness equipment company wants to understand how different customer segments purchase treadmill models in order to improve product targeting and marketing strategy.

## Dataset
The dataset includes customer demographics, income, fitness level, weekly usage, and treadmill model purchased.

## Key Analysis Questions
- Which customer segments buy each treadmill model?
- Do income and fitness levels influence product choice?
- Are premium models purchased by higher-income or more active customers?
- Is usage frequency linked to model type?
- Are certain products over- or under-targeted?
- Which products should be marketed to which demographics?

## Tools Used
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Jupyter Notebook

## Key Insights
- TM798 is predominantly purchased by high-income, highly active customers
- TM195 and TM498 serve overlapping entry and mid-tier customer segments
- Usage frequency and weekly miles increase with model tier

## Recommendations
- Position TM798 as a premium lifestyle product
- Market TM195 and TM498 as accessible and value-driven options
- Consider clearer differentiation between entry and mid-tier models

## Next Steps
Future analysis could include behavioural or marketing data to assess cross-selling and long-term customer value.
